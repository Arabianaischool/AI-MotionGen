البدء السريع | QUICK START
================================================================================

1. انقر مرتين على ملف "start-ui.bat"
   Double-click "start-ui.bat"

2. ستفتح الواجهة في المتصفح تلقائيًا
   The interface will open in your browser automatically

3. اكتب أو الصق كود TSX في محرر الكود
   Write or paste TSX code in the editor

4. اضغط زر "توليد الفيديو"
   Click "توليد الفيديو" (Generate Video) button

================================================================================
متطلبات الكود | CODE REQUIREMENTS
================================================================================

يجب أن يحتوي كود TSX على:

1. تصدير compositionConfig:
   export const compositionConfig = {
     id: 'MyVideo',
     durationInSeconds: 5,
     fps: 30,
     width: 1080,
     height: 1920,
   };

2. تصدير المكون الرئيسي:
   export default MyComponent;

================================================================================
استكشاف الأخطاء | TROUBLESHOOTING
================================================================================

"فشل في حفظ الملف":
  - تأكد من أن المجلد غير محمي للكتابة
  - Make sure the folder is not write-protected

"فشلت العملية":
  - تحقق من كود TSX للأخطاء
  - Check your TSX code for syntax errors
  - تأكد من تصدير compositionConfig

"لم يتم إنشاء ملف الفيديو":
  - قد تكون هناك مشكلة في الكود
  - راجع سجل الأخطاء في الواجهة

"EADDRINUSE" خطأ:
  - السيرفر شغال من قبل، أغلق النافذة الأولى وافتح تاني
  - Server is already running, close the first window and try again

في أول تشغيل:
  - سيتم تحميل متصفح Chromium (~150MB)
  - هذا التحميل لمرة واحدة فقط
